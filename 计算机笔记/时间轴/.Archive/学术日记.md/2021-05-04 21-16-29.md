- 怎么把本地仓库同步到 github 上？
- 答：
1. cd 进入要同步的 git 本地仓库文件夹
2. git add . (.表示全部放入本地)
3. git commit  -m “XXX” (放入暂存区)
4. git push (提交到云端)
